# Post-Change Checklist

Run this checklist after every development task before marking the work complete.

## Checklist

- [ ] Used Platinum API only
- [ ] No local DB added
- [ ] No fallback data added
- [ ] Business logic kept out of UI
- [ ] Typed DTOs updated
- [ ] Angular-ready structure preserved
- [ ] Deployment-ready structure preserved
- [ ] Errors/loading states handled
- [ ] Dead code removed
- [ ] Compliance review completed

## Required Process After Every Task

1. Implement the change
2. Review affected files
3. Check for any local storage/database usage
4. Check for any mock or fallback data
5. Check Angular-readiness
6. Check deployment-readiness
7. Report compliance result
